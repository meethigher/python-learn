# Python 标准库示例集合

以下为示例文件列表：

001_argparse_demo.py - 命令行参数解析示例
002_asyncio_demo.py - asyncio 基本示例
003_collections_demo.py - namedtuple, defaultdict, Counter
004_concurrent_futures_demo.py - 线程池/进程池示例
005_contextlib_demo.py - contextmanager 演示
006_csv_demo.py - CSV 读写
007_datetime_demo.py - 日期时间操作
008_functools_demo.py - lru_cache, partial
009_http_server_demo.py - 静态文件服务器示例（基于 http.server）
010_itertools_demo.py - itertools 常用函数
011_json_demo.py - JSON 编解码
012_logging_demo.py - logging 配置与示例
013_multiprocessing_demo.py - 进程示例
014_os_demo.py - 文件/目录操作
015_pathlib_demo.py - Path 用法
016_socket_echo_server.py - TCP 回显服务器示例（示例文件，运行会绑定端口）
017_sqlite3_demo.py - sqlite3 使用示例
018_subprocess_demo.py - 启动外部进程示例
019_sys_demo.py - sys.argv / sys.exit
020_tempfile_demo.py - 临时文件/目录示例
021_threading_demo.py - 线程示例
022_typing_demo.py - typing 注解示例
023_unittest_demo.py - unittest 基本用法示例
024_urllib_demo.py - urllib.request 简单请求示例

使用方式：
解压后运行 `python 文件名.py`（测试环境 3.11.9 ）。
